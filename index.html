Solving the Vehicle Scheduling Problem with Reinforcement Learning

Technologies: Tensorflow, GNN

Operational costs in public bus transportation are mainly influenced by personnel and bus operations. Therefore, planning needs to be as accurate as possible. Usually solved with heuristcs, my Master thesis uses reinforcement learning instead - an promising approach that has not been tried in research so far.

Focusing on the Vehicle Scheduling Problem, the RL-Agent might be able to generalize problems which leads to an advantage over heuristics which need repetitive customizations. Work is currently in progress with a submission date in July.

Using Computer Vision to Measure Hands

Technologies: OpenCV

A common project of a friend and me that we have worked on during the first lockdown caused by the pandemic. Our goal was to learn more about Computer Vision because we were both fascinated by this technology, but only had a vague idea about its implementation. Using openCV and various mathematic libraries, we developed a tool that is able to measure a hand simply by using a coin as a reference.

We realized that poor lighting and a noisy background inhibits precise results. Nevertheless, our code is able to provide measurements with a usual deviation of only +-3mm.
